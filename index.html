소재 준비
MD 파일로 영상 소재를 작성합니다.
기존 콘텐츠(실전_Tips, 학습용_Books 등)를 그대로 사용하거나
새로운 주제를 직접 작성합니다.
사용자 직접 작성
입력: input/topic.md
1
경쟁 영상 리서치
소재와 유사한 인기 유튜브 영상을 자동 검색합니다.
Gemini CLI가 인기 영상 3개를 찾고,
yt-dlp + youtube-transcript-api가 자막과 메타데이터를 직접 추출합니다.
Gemini는 추출된 자막을 받아 구조 분석 + 뼈대 생성만 합니다.
Gemini CLI + yt-dlp + youtube-transcript-api
출력: output/research/[타임스탬프]/
  metadata.json transcript.txt analysis.txt skeleton.txt
2
올인원 스크립트 생성
소재 + 리서치 결과를 참고하여
제목 + 썸네일 문구 + 전체 대본을 한번에 생성합니다.
사용자 확인 후 수정 가능합니다.
Claude CLI
출력: output/scripts/[타임스탬프]/
  title.txt thumbnail.txt script.txt
3
스크립트 분석
확정된 대본을 분석하여 섹션별로 분리합니다.
각 섹션의 예상 시간, 키워드, 제목 힌트를 추출합니다.
Python (script_analyzer)
내부 처리 → ScriptAnalysis 객체 생성
4
이미지 프롬프트 + 음성 생성
섹션별 이미지 생성용 프롬프트를 만들고,
ElevenLabs API로 TTS 음성 파일을 생성합니다.
첫 번째 이미지 프롬프트는 클립보드에 자동 복사됩니다.
ElevenLabs API + Python
출력: 03_prompts/image_prompts.txt
       01_voice/section_*.mp3
5
Vrew 가이드 + 메타데이터 생성
Vrew 영상 편집을 위한 타임라인 가이드와
유튜브 업로드용 메타데이터(제목, 설명, 태그)를 자동 생성합니다.
Python (guide + metadata generator)
출력: 04_guide/vrew_guide.txt
       05_metadata/metadata.txt
6
Vrew 편집 → 유튜브 업로드
생성된 음성 + 이미지 + 가이드를 Vrew에서 조합하여
영상을 완성하고 유튜브에 업로드합니다.
메타데이터를 복사하여 제목/설명/태그를 입력합니다.
Vrew (수동) + YouTube
결과: 유튜브 영상 URL
7
블로그 글 생성 (선택)
유튜브 URL을 입력하면 영상 대본을 기반으로
블로그 글을 자동 생성합니다.
Python (blog_generator)
출력: 06_blog/blog.md

CLI 명령어

# 스킬 실행 (전체 프로세스 올인원)
/youtube-generate input/topic.md

# 또는 단계별 수동 실행:
python orchestrator.py --research input/topic.md          # Step 1: 리서치
python orchestrator.py --write input/topic.md             # Step 2: 대본 생성
python orchestrator.py output/scripts/.../script.txt       # Step 3~5: 재료 생성
python orchestrator.py output/scripts/.../script.txt --short # 숏폼 모드
python orchestrator.py --blog <유튜브URL> output/[폴더]   # Step 7: 블로그
